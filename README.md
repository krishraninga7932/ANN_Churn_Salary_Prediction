# Salary Regression Prediction using Artificial Neural Network (ANN)

## Overview

This project is a Salary Regression Prediction web application built using TensorFlow, Scikit-Learn, and Streamlit. The model predicts a customer's estimated salary based on various banking-related attributes such as credit score, geography, age, balance, tenure, number of products, and account activity.

The project demonstrates how Artificial Neural Networks (ANNs) can be applied to regression problems and deployed as an interactive web application using Streamlit.

---

## Features

- Predict customer salary using a trained ANN model
- Interactive Streamlit user interface
- Real-time salary prediction
- Data preprocessing using:
  - Label Encoding
  - One-Hot Encoding
  - Feature Scaling
- Trained regression model saved and loaded using TensorFlow
- User-friendly input forms

## Dataset

The model is trained using the **Churn Modelling Dataset**, which contains customer information such as:

- Credit Score
- Geography
- Gender
- Age
- Tenure
- Balance
- Number of Products
- Credit Card Status
- Active Membership Status
- Exited Status
- Estimated Salary

For this project, **EstimatedSalary** is used as the target variable for regression.
